# token-reduction-ultimate

The ultimate global skill to maximize token savings using the full SuperCLI ecosystem.

Combines rtk, tokf, context-mode, zagi, token-optimizer-cli, codefetch, project-map, yek, probe, colgrep, codedb, slowql, sloc-guard, chainlink, and token-count into a unified decision-driven workflow.

Covers discovery via `sc bin`, installation, smoke-testing, and daily usage patterns proven to save 60-98% on token consumption.

## Install as SuperCLI Plugin

```bash
sc plugins install token-reduction-ultimate --json
sc skills get token-reduction-ultimate:main.skill
```
